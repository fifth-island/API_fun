Assignment Details


API selected:   https://zoo-animal-api.herokuapp.com/


This API was located in the public library of APIs on Github provided in the resources for the assignment.

Source:   https://github.com/public-apis/public-apis


API's description:   This API provides facts and pictures about random animals. It is a public API where users can request zoo animal data in JSON format. This API supports two possible endpoints.
   -> The first returns a random animal and all its information (name, latin name, animal type, active time, length and weigth's ranges, lifespan, habitat, diet, location, ID, and image).
   -> The second endpoint returns a list of animal structs, being able to return from 1 to 10 at the same time.


API's options used:   The request focused just in the first endpoint. Every time the user click on the button, a new random animal is retrieved from the API database and displayed in the screen. For this assigment, the elements used were "name", "origin", and "image", which helped to compose a descriptive introduction to the "rescued animals."


API's applicability:   This API could be user for didatic purposes.
   -> It could be used for teaching children about ecology, animal names, and all diversity in the world.    -> It could be used as the database for some virtual game. The possibilities of games to be made with this API are vast, considering that memory games, guess the animal, and several other games realting to memory and general knowledge could use of this API.
